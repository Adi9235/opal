# Run OPAL as Python packages (python 3.7 >) / CLI utilities

## CLI execution
### Server
 - install
    ```sh
    pip install opal-server
    ```
- run
    ```sh
    # get CLI help
    opal-server --help
    # run server as daemon
    opal-server run
    ```

### Client
- install
    ```
    pip install opal-client
    ```
- run
    ```sh
    # get CLI help
    opal-client --help
    # run the client as a daemon
    opal-client run
    ```
